🧠 LeetPattern

Master coding interview patterns, not just problems.

🚀 Overview

LeetPattern is a modern flashcard-based learning platform designed to help students prepare for coding interviews efficiently. Instead of memorizing hundreds of problems, users learn the core patterns behind them through interactive flashcards.

✨ Features

- 🎴 50+ LeetCode Pattern Flashcards
- 🔍 Instant Search
- 🏷️ Category Filters
- 🔄 Interactive Flip Cards
- 🐍 Python Code Templates
- 💡 Quick Interview Tricks
- ❤️ Favorite Flashcards
- 🎲 Random Pattern Generator
- 🌙 Dark Mode
- 📱 Fully Responsive Design
- 💾 Local Storage Support
- ⚡ Fast and Lightweight
- 🚀 GitHub Pages Ready

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub Pages

🎯 Purpose

This project helps beginners and interview aspirants quickly revise the most frequently asked coding interview patterns from companies like Amazon, Google, Microsoft, and Meta.
